## 测试题 916：最长公共子序列（LCS）

**测试题考点**：二维 DP，`dp[i][j]` 表示前缀 LCS 长度

| 洛谷题号 | 题目 | 难度 | 对应点 |
|:---:|:---|:---:|:---|
| P1439 | 【模板】最长公共子序列 | 普及- | **模板原题**，直接练 LCS 核心代码 |
| P2758 | 编辑距离 | 普及/提高- | LCS 变式，三种操作的状态转移 |
| P1438 | 绿豆蛙的归宿 | 普及/提高- | 最长单调子序列，LCS 的 O(n log n) 优化思路（进阶） |

**复习重点**：LCS 转移方程是期末大题重灾区，务必能手写：
```java
if (X[i-1] == Y[j-1]) dp[i][j] = dp[i-1][j-1] + 1;
else dp[i][j] = Math.max(dp[i-1][j], dp[i][j-1]);
```

## 测试题 917：最长回文子序列（LPS）

**测试题考点**：区间 DP，`dp[i][j]` 表示子串 `[i,j]` 的 LPS 长度，注意遍历顺序

| 洛谷题号 | 题目 | 难度 | 对应点 |
|:---:|:---|:---:|:---|
| P1435 | [IOI2000] 回文字串 | 普及/提高- | 回文串 DP 经典，LPS / 最少插入次数 |
| P2858 | [USACO06FEB] 奶牛回文 | 普及/提高- | 回文子序列 / 子串练习 |
| P1015 | [NOIP1999] 回文数 | 普及- | 回文判定 + 高精度（辅助理解回文性质） |

**复习重点**：LPS 和 LCS 的区别：
- LCS：`dp[i][j]` 依赖左上、上方、左方 → i 从小到大，j 从小到大
- LPS：`dp[i][j]` 依赖左下、下方、左方 → **i 从大到小**，j 从小到大
